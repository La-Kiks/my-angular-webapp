Portfolio

Mon site vitrine, qui permet de présenter mes projets ainsi que de m'entrainer à coder.

Dans ma quête d'améliorer mon php j'ai voulu créer ce site, similaire à un blog. Le site a vocation à représenter mon travail de développeur en priorité mais il est voué à évoluer. Le but étant de me familiariser avec Symfony, je teste un maximum de fonctionnalités en backend même si je ne les utilise pas toutes.

Par la suite j'ai refait le site sous Angular afin d'en apprendre plus sur ce framework et retravailler mes bases HTML/CSS/JS. Avec un backend .NET (en cours de création) pour créer une partie blog.

Technos : Angular, .NET, Symfony 7 (Pour la v1 du site).
